# NestJS-API-Server

根据官方文档以及各种网上资料整合,一套轻量级nestjs-api脚手架。

## Getting Start
``` javascript
 git clone https://github.com/BoBoooooo/NestJS-API-Server.git
 npm install 
 npm start
```
## Features

* 登录模块、jwt 鉴权

* socket.io 

* 高级查询

* BaseController BaseService

* 全局异常处理

* log

* 统一api ResultBean封装

* 文件上传,静态资源服务
